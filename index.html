Proyecto Ajedrez en WPF

Desarrollé un juego de ajedrez interactivo en WPF con C# y XAML, inspirado en plataformas como Lichess. El proyecto incluye una ventana de registro de jugadores para capturar sus nombres antes de la partida e implementa modos de juego Blitz, Bullet y Rapid, cada uno con temporizadores específicos. Usando los principios de la programación orientada a objetos, como la encapsulación para asegurar la integridad de los datos, la abstracción para definir comportamientos comunes, la herencia para reutilizar código y el polimorfismo para gestionar las acciones de las piezas, el juego valida los movimientos y reglas del ajedrez, incluyendo captura, jaque y jaque mate. Además, ofrece una interfaz altamente interactiva y fácil de usar, con botones para seleccionar jugadas, mostrar el estado de la partida y gestionar el control del tiempo, todo ello manteniendo una experiencia visual atractiva y fluida.

Proyecto Calculadora Cientifica en WPF

Desarrollé una calculadora científica en WPF utilizando C# y XAML, que permite realizar operaciones matemáticas básicas y avanzadas, como trigonometría (seno, coseno, tangente), logaritmos naturales, factoriales, potencias, raíces cuadradas y más. Implementé una interfaz sencilla para el usuario interactiva con botones, cuadros de texto y un historial de cálculos, además de opciones para cambiar entre grados y radianes. El proyecto fue diseñado con enfoque en la usabilidad y validación de entradas, asegurando un rendimiento eficiente y una experiencia intuitiva para el usuario.
